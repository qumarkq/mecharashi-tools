專武天賦強化顯示DiffHighlight · SkillCondition · SkillEffect

Problem
問題背景
問題一：專武對天賦的強化效果無法可視化
機師天賦頁目前僅顯示「最大強化」前後的差異（DiffHighlight），但許多專屬武器會額外強化特定天賦的效果（如調整係數、解鎖新動作）。 目前這些強化效果只能在「強化天賦」區塊讀到武器技能的文字描述，使用者無法直觀比較「有無專武時天賦的差異」。
問題二：WeaponSkill 缺少強化後天賦文字欄位
現有 WeaponSkill.enhancesTalentName 只記錄被強化的天賦名稱，沒有對應的「強化後天賦文字」欄位， 導致 UI 無法執行 DiffHighlight 差異對比，管理員也缺乏明確的填值目標。
問題三：SkillCondition 缺少 buff 狀態和移動狀態的條件型別
部分專武效果需要判斷「持有特定 buff 才生效」（如持有「強化射擊」時）或「本回合未移動才生效」。 現有 SkillCondition 缺少 hasBuff 欄位和 notMoved trigger 值， 計算器無法程式化處理這些條件，管理員也無法在 Admin 中結構化填入。
問題四：SkillEffect 無法表達「覆蓋型」效果
部分專武強化會覆蓋天賦的原始數值（如技能倍率 0.15 → 0.2），而非加算。 現有 SkillEffect 只有 value 欄位，計算器無法區分加算 vs 覆蓋， 若以加算方式處理覆蓋型效果，計算結果將錯誤。
Design
設計決策
決策一：在 WeaponSkill 新增 enhancedTalentDescription 欄位
強化後天賦文字由武器端持有，天賦結構不動。PilotDetailPage 在渲染天賦時， 查找是否有對應的 enhancedTalentDescription，若有則顯示「▶ 專武強化」DiffHighlight 區塊（黃色，區別於最大強化的青色）。

以 descriptionMax（最大強化版本）為比較基準，呈現的是「最大強化之後，再有專武能得到什麼額外差異」。
決策二：提升 usePilotExclusiveWeapon 至頁面層級
原本 hook 在 ExclusiveWeaponPanel 內部呼叫，天賦渲染區無法取得武器資料。 將 hook 提升至 PilotDetailPage，建立 talentEnhancementMap: Map<talentName, enhancedText>， 讓天賦渲染區和武器面板共用同一份資料，避免重複 fetch。
決策三：SkillCondition 新增 hasBuff 欄位；trigger 新增 hasBuff / notMoved / onCrit
· hasBuff?: string：配合 trigger: 'hasBuff'，填入需持有的狀態名稱
· trigger: 'notMoved'：本回合未移動（無額外參數）
· trigger: 'onCrit'：造成暴擊後觸發
Admin 的 SkillConditionEditor 同步新增對應的輸入欄位。
決策四：SkillEffect 新增 valueType 欄位
valueType?: 'add' | 'override'，預設為 'add'（加算，向後相容）。 填入 'override' 時，計算器以此值取代對應天賦效果的原始值，而非加算。 Admin 的 SkillEffectItem 同步新增下拉選單。
決策五：遊戲原文不動
PilotTalent.description 與 descriptionMax 保持遊戲原文，不允許為了差異對比而修改。 WeaponSkill.enhancedTalentDescription 亦填遊戲內讀到的強化後天賦描述原文。 Admin 填值指引強調「複製遊戲原文，不得改寫措辭」。
Example
凱莉專武「昏星繼光」填值示例
// WeaponSkill — 白鶴之眺（carry 生效） { name: "白鶴之眺", type: "被動技能", activation: "carry", description: "強化[白鶴]的效果：擁有[強化射擊]時...", // 武器技能遊戲原文，不動 enhancesTalentName: "白鶴", enhancedTalentDescription: "擁有[強化射擊]時，攻擊時技能倍率提升從0.15提升至0.2，...", // 天賦強化後原文 effects: [ { stat: "skillCoefficient", value: 0.2, valueType: "override", // 覆蓋原始值 0.15 scope: "self", condition: { trigger: "hasBuff", hasBuff: "強化射擊" } }, { stat: "critRate", value: 15, scope: "self", condition: { trigger: "notMoved", weaponKind: "SNIPER" } } ], buffIds: [] }
