A-2   GameDataContext 讀取流程加入版本比對
src/contexts/GameDataContext.tsx
待開始
useEffect 啟動時先讀 meta/gameData，命中快取則跳過 11 次讀取
修改 GameDataContext.tsx 的 useEffect：
1. getDoc(doc(db, 'meta', 'gameData'))（1 read）
2. 比對 readGameDataCache().version
3. 版本相同 → setState from cache，setLoading(false)，return
4. 版本不同 → 走原 Promise.all 讀取流程，完成後 writeGameDataCache
5. meta/gameData 不存在（remoteVersion = null）→ 直接走原流程，不寫 localStorage

src/contexts/GameDataContext.tsx
待開始
reload() 呼叫前先清除 localStorage 快取，確保強制重讀行為正確
