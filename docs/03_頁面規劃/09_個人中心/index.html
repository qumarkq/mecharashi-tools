我的配裝 Tab
版面與互動
項目 說明
配裝列表 從 Firestore users/{uid}/builds 讀取，依 updatedAt 降序排列
配裝卡片 顯示：配裝名稱、機師名稱、機甲名稱、武器 ID、最後更新日期
載入配裝 點擊「載入配裝」按鈕，使用 React Router navigate state 將配裝資料帶至 /simulator，並直接跳到結果步驟
刪除配裝 點擊垃圾桶圖示，確認後呼叫 deleteBuild(uid, buildId) 刪除
空狀態 顯示引導前往 /simulator 新增配裝的提示
頭像來源邏輯
AvatarDisplay 顯示規則
avatarType 顯示內容 資料來源
null（預設） 顯示名稱首字母彩色圓形（依名稱 hash 決定顏色） — （本地計算）
'upload' 用戶上傳的圖片 Firebase Storage avatars/{uid}/avatar.webp，URL 存於 avatarUrl
'pilot' 選取機師的頭像 從 pilots collection 即時取得（avatarPilotId 對應機師的 portrait）
'google' Google 帳號頭像 Firebase Auth photoURL（第三方登入用戶）
