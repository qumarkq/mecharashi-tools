版面策略

左側：分步選配面板（Step 1~8）。右側：配裝預覽卡片（即時更新）。行動版：上下分割或 Tab 切換。

⚡

自動篩選提示

選機甲後，武器選單自動過濾超重選項，背包選單過濾不相容裝甲類型。不可選的項目變灰但保留。

📸

匯出分享圖（Phase 3）

html2canvas 截圖 → PNG。1200×675px 分享卡片，含機師/機甲/武器/背包/模組/元件資訊。傷害欄位預留。

🔖

配裝存檔（Phase 5）

登入用戶點擊「儲存配裝」後存入 Firestore users/{uid}/builds，可命名並在個人中心查看。

⚔️

傷害計算（Phase 8）

填入敵方參數後顯示期望傷害，支援 A/B 組合對比。傷害來源拆解面板。
