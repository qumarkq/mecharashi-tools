模組卡片

模組圖示、名稱、稀有度標籤（S/A）、槽位標籤（特性/8級/通用/副模組）。有等級資料時右上角顯示「◉ 等級效果」指示器；固定時顯示「📌」。描述文字中的數字與 % 符號以紅色粗體標示（✅ 已實作：highlightNumbers()）。

📊

各等級效果彈窗

懸停卡片時浮現等級彈窗，逐 Lv 顯示描述文字與數值（dmg/暴擊/爆傷/命中等）。點擊卡片可固定彈窗（📌），再次點擊或點擊背景可關閉。描述文字中的數字與 % 符號以紅色粗體標示（✅ 已實作：highlightNumbers()）。

📍

數值標籤 & 來源

卡片底部顯示最高等級非零數值標籤（傷害/暴擊/爆傷/命中/火力/護甲/出力/回避/耐久/減傷/抗暴）。來源欄位顯示 source 陣列，拆機甲來源另標注對應機甲名稱。
