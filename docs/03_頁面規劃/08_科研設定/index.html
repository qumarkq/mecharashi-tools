機師科研（依職業）

六個職業（突擊手/狙擊手/戰術家/格鬥家/守護者/機械師）各自有等級輸入欄，即時計算對應加成。

🤖

機甲科研（依裝甲類型）

輕型/中型/重型三種裝甲類型，各自有等級輸入，顯示耐久/護甲/火力等加成預覽。

🔫

武器科研（依武器種類）

各武器種類（刀劍/拳套/重機槍…）等級輸入，顯示攻擊力/命中/爆率加成預覽。

📋 儲存機制 · 已登入：設定自動存入 Firestore users/{uid}/profile.globalResearch，下次開啟自動帶入。
· 未登入：設定暫存於 localStorage，清除瀏覽器資料後遺失。
· 設定頁提示用戶登入以永久儲存。
🔬
