頁面區塊
🔫

武器 Hero 區塊

武器名稱、稀有度標籤（SS / S+ / S / A / B）、大類 / 種類標籤、機甲限制標籤（無限制 / 輕型 / 中型 / 重型）、裝備部位標籤（單手 / 雙手 / 肩膀 / 背後）。

📊

基礎屬性面板

攻擊力（含子彈數，如 766×1）、命中值、暴擊值、重量、射程（manhattan / orthogonal 顯示 "2-4"；ring 顯示 "2+"）、彈藥量、連擊數。

✨

技能列表

每個技能卡片顯示：生效方式標籤（攜帶=紫 / 裝備中=青 / 使用時=橘）、技能名稱、文字描述、結構化效果數值。
若技能有 enhancesTalentName，顯示「強化天賦」區塊（黃色 DiffHighlight）。

🔧

改裝資訊

固定改裝：方案名稱、可改等級上限、固定效果列表（stat + value）。
浮動改裝：方案名稱、可填欄位數、所有可能效果（含觸發條件與 min ~ max 範圍）。

⚡

元件插槽資訊

觸元件插槽數 / 應元件插槽數、元件上限（componentLimit）顯示。以小格子圖示呈現，空格可視為可用槽位。

⭐

專武資訊（isExclusive=true）

顯示對應機師名稱及連結（exclusiveFor）；技能區塊會額外呈現天賦強化 DiffHighlight，讓使用者直觀看到「有此專武時天賦的強化版本」。
