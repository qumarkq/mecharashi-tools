01 — Overview
枚舉設計原則
所有分類欄位改用 TypeScript Enum 管理，Firestore 中儲存 enum 的字串值（value）。
📁 統一存放於 src/types/enums.ts 前端所有元件統一 import 此檔案的 enum，不再使用散落各處的字串字面量（string literal）。

02 — Code
完整 Enum 定義範例
src/types/enums.tsexport enum PilotClass { ASSAULT = '突擊手', SNIPER = '狙擊手', TACTICIAN = '戰術家', FIGHTER = '格鬥家', GUARDIAN = '守護者', ENGINEER = '機械師', CONSTRUCTOR = '調構師', } export enum ModuleSlot { SLOT_4 = '機甲特性模組', SLOT_8 = '機甲8級模組', UNIVERSAL = '通用模組', BUILT_IN = '機甲副模組', } export enum ItemRarity { EX = 'EX', // 僅專武使用 S = 'S', A = 'A', B = 'B', } export enum DataManaged { SCRIPT = 'SCRIPT', MANUAL = 'MANUAL', } // ... 其餘 enum 依同樣模式定義
