01 — Security
存取控制
🔐 用戶資料安全規則 users/{uid} 及其所有子 Collection 僅本人（isOwner(uid)）可讀寫，其他用戶與管理員均無法存取。 詳細 Firestore Security Rules 請見 安全規則文件。

03 — builds
users/{uid}/builds/{buildId} — 用戶配裝
🗃️ users/{uid}/builds/{buildId} — 用戶配裝 USER DATA
欄位型別說明
buildNamestring配裝名稱

mechIdstring→ mechs collection
