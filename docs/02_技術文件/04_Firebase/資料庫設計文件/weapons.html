🗃️ weapons — 武器資料
欄位型別說明
idstring武器識別碼
namestring武器名稱
categoryWeaponCategory武器大類 Enum：射擊 / 格鬥 / 突擊 / 戰術
typestring武器種類（機槍 / 狙擊步槍 / 重機槍 / 拳套 ...）
typeCoefficientnumber武器種類係數（如狙擊步槍 = 5.92）
attackstring攻擊力（如 "766×1"）
accuracynumber命中值
critValuenumber暴擊值
weightnumber重量
rangestring射程（如 "2-4"）
rarityItemRarity稀有度 Enum：EX / S / A / B（EX 僅用於專武）
isExclusiveboolean是否為專屬武器
exclusiveForstring?主要對應機師 ID → 引用 pilots collection
triggerSlotsnumber觸元件插槽數量
effectSlotsnumber應元件插槽數量
▎fixedMod — 固定改裝方案
 ├ planNamestring方案名稱
 ├ maxLevelnumber最高等級
 └ effects[]array效果列表 { stat, value }
▎floatingMod — 浮動改裝方案
 ├ planNamestring方案名稱
 ├ slotsnumber可選槽位數量
 └ possibleEffects[]array可選效果列表 { stat, condition, value }
▎skills[] — 武器技能（嵌入式）
 ├ namestring技能名稱
 ├ activationSkillActivation生效時機 Enum：carry / equip / use
 ├ typeSkillType技能類型 Enum：被動技能 / 主動技能
 ├ descriptionstring技能效果描述
 ├ dmg / crit / critDmg / accnumber可量化加成數值
 └ buffIds[]string[]?關聯 BUFF ID 列表 → 引用 buffs collection
