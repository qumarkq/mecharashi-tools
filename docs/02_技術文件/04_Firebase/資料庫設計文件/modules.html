🗃️ modules — 模組資料
欄位型別說明
idstring唯一識別碼（特性/8級/通用：mod_{masterApiId}；副模組：mod_{機甲名}_{position}）
namestring模組名稱
slotModuleSlot槽位 Enum：機甲特性模組 / 機甲8級模組 / 通用模組 / 機甲副模組
boundMechIdstring?綁定機甲 ID → 引用 mechs collection；null = 可獨立取得
boundPartMechPart?綁定部位 Enum：torso / leftArm / rightArm / legs；null = 無綁定
availableboolean可獨立取得：特性/8級/通用 = true；副模組 = false
sourceModuleSource遊戲取得途徑 Enum：商店 / 拆機甲 / 未知
managedByDataManaged資料維護方式 Enum：SCRIPT / MANUAL
output_bonusnumber機甲出力增加（預設 0）
dmgnumber增傷加成 %（最高等級）
crit_ratenumber暴擊率加成（最高等級）
critDmgnumber暴擊傷害加成 %（最高等級）
acc_ratenumber命中率加成（最高等級）
firepower_ratenumber火力提升 %（最高等級）
armor_ratenumber護甲提升 %（最高等級）
crit_resist_ratenumber被暴擊率降低 %（最高等級）
dodge_ratenumber回避率 %（最高等級）
durable_ratenumber耐久值提升 %（最高等級）
dmg_resist_ratenumber遭受傷害降低 %（最高等級）
descriptionstring最高等級效果描述
rarityItemRarity稀有度 Enum：S / A / B（模組不使用 EX）
iconstring?圖示路徑
buffIds[]string[]?關聯 BUFF ID 列表 → 引用 buffs collection
▎levels[]（特性/8級/通用模組有各等級資料；副模組為空陣列）
 ├ levelnumber等級上限（該筆資料的最高等級，如 4 代表此為 4 級時的數值）
 ├ descriptionstring該等級效果描述
 ├ dmgnumber該等級增傷加成（預設 0）
 ├ crit_ratenumber該等級暴擊率加成
 ├ critDmgnumber該等級暴擊傷害加成
 ├ acc_ratenumber該等級命中率加成
 ├ firepower_ratenumber該等級火力提升 %
 ├ armor_ratenumber該等級護甲提升 %
 ├ crit_resist_ratenumber該等級被暴擊率降低 %
 ├ output_bonusnumber該等級機甲出力增加
 ├ dodge_ratenumber該等級回避率 %
 ├ durable_ratenumber該等級耐久值提升 %
 └ dmg_resist_ratenumber該等級遭受傷害降低 %
