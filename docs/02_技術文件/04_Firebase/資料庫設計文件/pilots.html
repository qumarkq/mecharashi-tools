🗃️ pilots — 機師資料
欄位型別說明
idstring唯一識別碼（如 "pilot_001_葉夫根尼"）
namestring顯示名稱
fullNamestring完整全名
rarityItemRarity稀有度 Enum：S / A / B（機師不使用 EX）
classPilotClass職業 Enum：突擊手 / 狙擊手 / 戰術家 / 格鬥家 / 守護者 / 機械師 / 調構師
factionstring所屬勢力
licenseMechLicense駕駛許可 Enum：輕型 / 中型 / 重型
masterLevelstring駕駛等級（如 "大師Ⅲ"）
profileobject個人資料 { gender, bloodType, height, additionalInfo{} }
statsobject滿級六維 { melee, assault, shooting, tactics, defense, engineering }
statsBaseobject初始六維（同結構）
apobject滿級 AP { init, max, recovery }
apBaseobject初始 AP（同結構）
talents[]array天賦列表（嵌入式，見下方 SkillEntry 結構）
skills[]array技能列表（嵌入式，見下方 SkillEntry 結構）
neuralDrive[]array神經驅動列表（嵌入式，見下方 SkillEntry 結構）
attacknumber攻擊素質
defensenumber防禦素質
lorestring角色故事（Markdown）
portraitstring立繪圖片路徑
▎嵌入式 SkillEntry 結構（talents / skills / neuralDrive 共用）
 ├ namestring技能名稱
 ├ typeSkillType技能類型 Enum：被動技能 / 主動技能
 ├ descriptionstring技能描述
 ├ descriptionMaxstring?滿級描述（若與 description 不同）
 ├ iconstring?圖示路徑
 ├ dmg / crit / critDmg / accnumber可量化加成數值
 └ buffIds[]string[]?關聯 BUFF ID 列表（引用 buffs collection）
