🗃️ components — 元件資料
欄位型別說明
idstring元件識別碼
namestring元件名稱
slotComponentSlot槽類型 Enum：trigger（觸元件）/ effect（應元件）
levelnumber元件等級
probabilitystring觸發概率類型："概率" / "必定"
descriptionstring完整效果描述
rarityItemRarity稀有度 Enum
▎trigger 專用欄位（slot = TRIGGER）
conditionTypeConditionType觸發條件類型 Enum：dualWield / singleWield / firstAttack / apCost / targetTorso / always
conditionstring觸發條件描述文字
conditionValuenumber?條件參數值（如 AP 消耗數）
▎effect 專用欄位（slot = EFFECT）
effectTypeEffectType效果類型 Enum：dmgBoost / bulletAdd / multiplierBoost / armorBreak / apDmgBoost / torsoDmgBoost
valuenumber效果數值
