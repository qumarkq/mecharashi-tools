01 — pilotResearch
pilotResearch Collection 欄位定義
各機師個別科研選項，每位機師一份 Document，欄位對應三種特質 × I/II/III 等級的可選加成。
🗃️ pilotResearch — 機師個別科研
欄位型別說明
pilotIdstring對應機師 ID → 引用 pilots collection

mechTraits[]array機甲特質選項（同結構）
weaponTraits[]array武裝特質選項（同結構）
exclusionRules[]array互斥規則 { slotA, optionA, slotB, optionB }
02 — globalResearch
globalResearch Collection 說明
全域科研定義，涵蓋機師 / 機甲 / 武器 三大類加成。
📌 globalResearch Schema — 待補充 此 Collection 存放全域科研各等級的數值加成定義（機師類、機甲類、武器類）。
玩家在 users/{uid}/profile 中儲存自己的科研等級選擇，由前端對照此 Collection 計算實際加成效果。
Schema 詳細欄位待遊戲資料整理後補充。
