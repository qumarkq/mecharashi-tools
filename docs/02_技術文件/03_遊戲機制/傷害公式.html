傷害公式DAMAGE FORMULA

核心傷害公式、總攻擊展開、期望值計算、增傷來源匯整

根據試算表與 NGA 社群研究整理
核心公式
傷害計算公式
核心傷害公式 最終傷害 = (總攻擊 - 敵方裝甲) × 增傷係數 × 爆擊係數 × 最終傷害係數
總攻擊展開 總攻擊 = (武器基礎攻擊力 + 固定改裝攻擊 + 浮動改裝攻擊) × 火力係數 + 科研火力 + 改造火力) × 五維係數 × 敵等級係數 × 駕駛等級係數 · 火力係數：每 3650 點火力 = 武器基礎攻擊力 ×100%（70EX 等級基準）
· 五維係數：近似三次方曲線，中心點約 8200–8400
· 敵等級係數：三次方曲線，Lv0 無衰減，Lv55 衰減 12.5%，Lv71 衰減 25%
· 固定改裝攻擊：基礎改裝提供的固定攻擊加成
· 浮動改裝攻擊：高級改裝的隨機攻擊加成（含條件觸發加成）
傷害期望值 期望值 = 基礎傷害 × (1 + 增傷%) × [ 爆率% × (1 + 爆傷%) + (1 - 爆率%) ] × 命中率% × 連擊數 × 技能倍率
增傷來源
增傷% 來源匯總（v1.2）
增傷% 來源匯總 增傷% = 機甲模組（四模/八模/固定） + 應元件（增傷型） + 背包技能 + 武器技能（攜帶即生效/裝備中生效）
        + 武器改裝（浮動改裝條件觸發） + 機師技能 + 天賦 + 神經驅動 + 機師個別科研 + 全域科研（增傷項） 爆率%、爆傷%、命中% 亦各自匯總上述所有來源中對應的加成。
子彈數/倍率/破甲/AP增傷/軀幹增傷由元件額外計算。
來源影響項目生效條件
機甲模組（4模/8模/固定）增傷 / 爆率 / 爆傷 / 命中裝備對應模組即生效
應元件增傷 / 子彈數 / 倍率 / 破甲 / AP增傷 / 軀幹增傷觸元件觸發後生效
背包技能增傷 / 爆率 / 爆傷 / 命中裝備背包即生效
武器技能（carry）增傷 / 爆率 / 爆傷 / 命中攜帶武器即生效（副武器也算）
武器技能（equip）增傷 / 爆率 / 爆傷 / 命中目前裝備的武器才生效
浮動改裝（條件觸發）增傷 / 命中 / 爆率滿足條件（目標HP%/裝甲類型）
機師技能增傷 / 爆率 / 爆傷 / 命中技能使用後，回合內生效
天賦增傷 / 爆率 / 爆傷 / 命中被動，常時生效
神經驅動增傷 / 爆率 / 爆傷 / 命中依所選選項，常時生效
機師個別科研增傷 / 爆率 / 爆傷 / 命中依所選特質，常時生效
全域科研火力 / 攻擊力 / 命中 / 爆率依玩家科研等級計算加成
