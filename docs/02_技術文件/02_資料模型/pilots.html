▎技能列表 skills[] — PilotSkill（PLAN-001）
涵蓋仿生電腦各核心單元解鎖的所有技能：職業技能（職業單元）、主動/指令/必殺/被動/武器技能（核心單元）
 ├ namestring技能名稱
 ├ typeSkillType (Enum)技能類型：被動技能 / 主動技能 / 指令技能（Order，有 CD）
 ├ apstring?AP 消耗；主動 / 指令 / 必殺 / 武器技能適用
 ├ cdstring?冷卻回合數；指令技能才有，對應 API 的 CD 欄位（計算器不用此欄位，僅供顯示）
 ├ weaponstring?限定武器類別；主動技能中需特定武器者填入，對應 API matchingWeaponType
 ├ descriptionstring效果描述文字
 ├ icon / iconLocalstring技能圖示路徑
 ├ effectsSkillEffect[]可量化效果列表（空陣列 = 待填入）；計算器加總所有 active 技能的 effects
 └ buffIdsstring[]技能觸發的 Buff ID 列表（API BufCarried 以 "/" 分割轉換）
▎天賦列表 talents[] — PilotTalent（PLAN-001）
 ├ namestring天賦名稱
 ├ typestring技能類型（通常為「被動技能」）
 ├ descriptionstring初始（0–2 星）效果描述
 ├ descriptionMaxstring滿星（3–5 星）效果描述
 ├ icon / iconLocalstring技能圖示路徑
 ├ effectsSkillEffect[]可量化效果列表（空陣列 = 待填入）
 ├ enhancedEffectsSkillEffect[]?裝備專屬武器後的加強版效果；省略或空陣列表示無加強版
 └ buffIdsstring[]天賦觸發的 Buff ID 列表
▎神經驅動 neuralDrive[] — NeuralDrive（PLAN-001）
 ├ namestring神經驅動區塊名稱
 ├ iconstring圖示路徑
 ├ slotsstring[]插槽類型說明列表
 └ levels[] — NeuralDriveLevel（每個激活閾值）
   ├ levelnumber激活層級（1 起；α / β 分區最多 3 層，γ 分區最多 6 層）
   ├ minSumnumber最低激活算力值；α / β 通常為 1 / 4 / 7，γ 可達 1 / 4 / 7 / 10 / 13 / 16
   ├ effectstring效果描述文字
   ├ skillName / skillIcon / iconLocalstring此層被動技能名稱與圖示
   ├ effectsSkillEffect[]可量化效果列表（空陣列 = 待填入）
   └ buffIdsstring[]激活此層時觸發的 Buff ID 列表
lorestring角色故事（Markdown 格式）
portraitstring立繪圖片路徑 /images/pilots/{id}.png
