◆ Data Model

pilots機師資料 · PILOT DATA

六維屬性、AP 系統、技能、天賦、神經驅動、個別科研

Schema
欄位定義
欄位型別說明
idstring唯一識別碼
namestring機師顯示名稱
rarityItemRarity (Enum)稀有度：S / A / B（機師不使用 EX）
classPilotClass (Enum)職業：突擊手 / 狙擊手 / 戰術家 / 格鬥家 / 守護者 / 機械師 / 調構師
▎六維屬性 stats
 ├ meleenumber格鬥
 ├ assaultnumber突擊
 ├ shootingnumber射擊
 ├ tacticsnumber戰術
 ├ defensenumber防禦
 └ engineeringnumber工程
▎AP 系統 ap
 ├ initnumber初始 AP
 ├ maxnumberAP 上限
 └ recoverynumberAP 回覆量
licenseLicenseType (Enum)駕駛許可：輕型 / 中型 / 重型
masterLevelstring駕駛等級
attacknumber攻擊素質
defensenumber防禦素質
▎技能列表 skills[]
 ├ namestring技能名稱
 ├ dmgnumber增傷加成 %
 ├ critnumber暴擊率加成
 ├ critDmgnumber暴擊傷害加成 %
 └ accnumber命中率加成
▎天賦列表 talents[]（結構同 skills[]）
 ├ namestring天賦名稱
 └ dmg / crit / critDmg / accnumber加成數值
▎神經驅動 neuralDrive[]（結構同 skills[]）
 ├ namestring神經驅動選項名稱
 └ dmg / crit / critDmg / accnumber加成數值
lorestring角色故事（Markdown 格式）
portraitstring立繪圖片路徑 /images/pilots/{id}.png
