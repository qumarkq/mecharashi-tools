mechs機甲資料 · MECH DATA

部件詳細資料、出力/重量、副模組引用 — 僅儲存滿級（manji）數據

UPDATED v1.4
v1.4 更新
重構說明
🔄 v1.4 修正：部件詳細資料獨立 + 模組映射 parts 欄位從耐久數值升級為完整部件物件（含火力、護甲、命中/閃避、移動力、接口、重量、圖示等）。
機甲詳情頁採座艙式排版（3×3 格），中央顯示立繪，四角顯示各部件屬性卡片。
部件圖片路徑：public/images/mechs/{機甲名}/{position}.png（portrait / half / torso / leftArm / rightArm / legs）。
僅儲存 manji（滿級）數據，不再保留初始 base 數據。

欄位定義
欄位型別說明
idstring唯一識別碼
namestring機甲名稱
armorTypeArmorType (Enum)輕型 / 中甲 / 重型
firepowernumber火力值（軀幹滿級）
armornumber裝甲值（軀幹滿級）
evasionnumber閃避值（腿部滿級）
mobilitynumber移動力（腿部滿級）
weightnumber總重量
outputnumber出力（影響武器/背包裝備限制）
▎部件詳細資料 parts.{position}（torso / leftArm / rightArm / legs）
 ├ durablenumber耐久值
 ├ armornumber護甲值
 ├ firepowernumber火力值
 ├ weightnumber部件重量
 ├ interfacestring接口類型（如 "Ⅱ型接口"）
 ├ hitnumber?命中值（僅左右臂有）
 ├ dodgenumber?閃避值（僅腿部有）
 ├ movenumber?移動力（僅腿部有）
 ├ outputnumber?出力（僅軀幹有）
 ├ antiRiotnumber?抗暴（僅軀幹有）
 └ iconstring?部件圖示路徑 /images/mechs/{機甲名}/{position}.png
▎副模組引用（指向 modules Collection）
module4Idstring?特性模組 ID（4模組槽位）→ modules Collection；無特性模組的機甲可省略
module8Idstring?8級模組 ID（8模組槽位）→ modules Collection；無8級模組的機甲可省略
moduleFixedIdsstring[]固定副模組 ID 列表（可多個）→ modules Collection
▎圖片路徑
portraitstring完整立繪 /images/mechs/{name}/portrait.png
halfstring半身圖 /images/mechs/{name}/half.png
