backpacks背包資料 · BACKPACK DATA

背包技能、重量、機甲裝甲類型限制

NEW v1.1
Schema
欄位定義
背包是獨立裝備，有自己的技能效果與重量，部分背包有機甲裝甲類型限制（如「僅限輕型機甲裝配」）。
欄位型別說明
idstring唯一識別碼
namestring背包名稱（如 "侵獵者背包"）
typestring效果分類標籤（增傷/護甲/再攻擊/首攻…）
weightnumber重量（佔機甲出力）
slotstring裝配位置（如 "背部"）
mechRestrictionArmorType? (Enum)機甲限制：null = 無限制 / "輕型" = 僅輕甲 / "重型" = 僅重甲
▎背包技能 skill
 ├ namestring技能名稱（如 "侵獵者驅動·增傷"）
 ├ typestring技能類型（被動技能）
 ├ descriptionstring完整效果描述
 ├ dmgnumber增傷加成 %
 ├ critnumber暴擊率加成
 ├ critDmgnumber暴擊傷害加成 %
 ├ accnumber命中率加成
 └ specialEffects[]array?非數值效果（如 "移動方式變為漂浮"）
篩選邏輯
配裝模擬器中的篩選規則
📋 背包選擇篩選邏輯 1. mechRestriction 過濾：若背包限輕甲，選了中/重機甲的玩家看不到此背包。
2. 重量過濾：背包重量 ≤ 機甲剩餘出力（出力 - 武器重量）才顯示。
3. 以上兩個條件皆由前端自動過濾，不需使用者判斷。
