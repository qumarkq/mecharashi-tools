pilotResearch機師個別科研 · PILOT RESEARCH

機師特質 / 機甲特質 / 武裝特質 — 三欄位各三段 — 選項互斥規則

NEW v1.1
設計說明
個別科研系統概覽
每位機師有三種科研類別，每類別有 I / II / III 三個欄位，每個欄位可從多個選項中選一個安裝。部分選項之間存在互斥關係。
👤

機師特質 I / II / III

強化機師本身的屬性或技能效果，通常與機師職業特性相關。

🤖

機甲特質 I / II / III

強化搭載機甲的屬性（火力/裝甲/閃避等）或機甲相關加成。

🔫

武裝特質 I / II / III

強化武器的攻擊力、暴擊、命中等屬性或武器技能效果。

Schema
欄位定義
欄位型別說明
pilotIdstring對應機師 ID（關聯 pilots Collection）
▎機師特質 pilotTraits[]
 ├ slotnumber欄位編號：1（I）/ 2（II）/ 3（III）
 └ options[]array該欄位可選的能力列表 ↓
  ├ namestring能力名稱
  ├ descriptionstring效果描述
  └ dmg / crit / critDmg / accnumber可量化的加成數值
▎機甲特質 mechTraits[]（結構同 pilotTraits[]）
 └ slot / options[]-欄位 I / II / III，各含選項列表
▎武裝特質 weaponTraits[]（結構同 pilotTraits[]）
 └ slot / options[]-欄位 I / II / III，各含選項列表
▎互斥規則 exclusionRules[]
 ├ slotAstring欄位 A（如 "pilotTraits.1"）
 ├ optionAstring能力 A 名稱
 ├ slotBstring欄位 B
 └ optionBstring能力 B 名稱（與 A 互斥）
⚠️ 待補充 互斥規則的具體邏輯（slotA/slotB 格式、多重互斥的處理方式）待遊戲資料確認後補充。 模擬器 Step 5 中，選到互斥選項時需顯示提示。
