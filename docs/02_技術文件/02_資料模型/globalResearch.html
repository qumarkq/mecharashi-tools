globalResearch全域科研 · GLOBAL RESEARCH

機師科研（依職業）/ 機甲科研（依裝甲）/ 武器科研（依種類）— 玩家自行輸入等級

NEW v1.1
設計說明
全域科研系統概覽
全域科研因玩家不同而異，網站提供「科研設定」頁讓玩家手動輸入自己各項科研的等級，系統自動計算對應加成。
📋 玩家輸入機制 登入用戶的科研設定存於 Firestore users/{uid}/profile.globalResearch，下次登入自動帶入。
未登入用戶的科研設定暫存於 localStorage。
globalResearch Collection 定義的是加成的計算規則（定義表），不是玩家個人資料。
Schema — 機師科研
pilotResearchByClass（依職業分類）
欄位型別說明
classNamePilotClass (Enum)職業名（格鬥家/突擊手/狙擊手/戰術家/守護者/機械師）
▎固定值加成 flatBonus（滿等級時的數值）
 ├ meleenumber格鬥素質 +（如 +2244）
 ├ defensenumber防禦素質 +（如 +2244）
 ├ assaultnumber突擊素質 +（如 +1800）
 └ …number其他素質對應加成
▎百分比加成 percentBonus
 ├ meleenumber格鬥素質 × %（如 +20%）
 └ …number其他素質對應百分比加成
Schema — 機甲科研
mechResearchByType（依裝甲類型分類）
欄位型別說明
armorTypeArmorType (Enum)裝甲類型（輕型/中型/重型）
▎固定值加成 flatBonus（滿等級時）
 ├ torsoHPnumber軀幹耐久 +（如 +6029）
 ├ legHPnumber腿部耐久 +（如 +4529）
 ├ leftArmHP / rightArmHPnumber手臂耐久 +（如 +3035）
 ├ antiCritnumber抗暴 +（如 +2164）
 ├ evasionnumber閃避 +（如 +2544）
 ├ accuracynumber命中 +（如 +1698）
 ├ firepowernumber火力 +（如 +2944）
 ├ torsoArmornumber軀幹護甲 +（如 +294）
 └ legArmornumber腿部護甲 +（如 +196）
▎百分比加成 percentBonus
 ├ torsoHP / legHP / …number耐久 × %（如 +19%）
 └ firepower / antiCrit / …number火力/抗暴等 × %（如 +9%）
Schema — 武器科研
weaponResearchByType（依武器種類分類）
欄位型別說明
weaponTypestring武器種類（刀劍/拳套/重機槍…）
▎加成 bonus
 ├ attacknumber攻擊力 +（如 +462）
 ├ accuracynumber命中 +（如 +1900）
 └ critnumber暴擊 +（如 +2598）
