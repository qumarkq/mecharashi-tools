weapons武器資料 · WEAPON DATA

武器屬性、元件插槽、固定/浮動改裝、三種技能生效方式

UPDATED v1.2
v1.2 更新
重構說明
🔄 v1.2 修正：武器系統大幅擴充 武器有類型、元件插槽（觸元件/應元件數量不同）、固定改裝（確定效果）、浮動改裝（隨機範圍）。
專武不限制使用者，但技能會加強特定機師的天賦。
技能生效方式分三種：攜帶即生效（carry）/ 裝備中生效（equip）/ 僅使用時生效（use）。
Schema
欄位定義
欄位型別說明
idstring唯一識別碼
namestring武器名稱（如 "昏星繼光"）
categoryWeaponCategory (Enum)武器大類：射擊 / 格鬥 / 突擊 / 戰術
typestring武器種類（機槍/狙擊步槍/重機槍/拳套…）
typeCoefficientnumber武器種類係數（狙擊步槍 = 5.92）
attackstring攻擊力（如 "766×1"，含子彈數）
accuracynumber命中值
critValuenumber暴擊值
rangestring射程（如 "2-4"）
weightnumber重量（影響機甲出力限制）
rarityRarity (Enum)稀有度
isExclusiveboolean是否為專屬武器
exclusiveForstring?主要對應機師 ID（專武天賦加強對象）
▎元件插槽
triggerSlotsnumber觸元件插槽數量
effectSlotsnumber應元件插槽數量
▎固定改裝 fixedMod（效果固定，依武器等級解鎖）
 ├ planNamestring改進方案名稱（如 "機槍VIII"）
 ├ maxLevelnumber可改裝等級上限（如 70）
 └ effects[]array固定效果列表 { stat: "attack"/"crit"/"accuracy", value: number }
▎浮動改裝 floatingMod（效果隨機，有範圍）
 ├ planNamestring進階方案名稱（如 "機槍IV"）
 ├ slotsnumber可裝填效果欄位數
 └ possibleEffects[]array所有可能效果 ↓
  ├ statstring效果類型：attack / crit / accuracy / firepower
  ├ conditionstring?觸發條件：null（無條件）/ "targetHP<50%" / "vsLight" / "vsMedium" / "vsHeavy"
  ├ minnumber數值下限
  └ maxnumber數值上限
▎武器技能 skills[]
 ├ namestring技能名稱（如 "白鶴之眺"）
 ├ typestring技能類型（被動技能/主動技能）
 ├ activationSkillActivation (Enum)carry = 攜帶即生效 / equip = 裝備中生效 / use = 僅使用時生效
 ├ descriptionstring技能效果描述
 ├ dmg / crit / critDmg / accnumber可量化的加成數值
 └ enhancesTalentstring?加強的特定機師天賦名稱（專武用）
💡 固定改裝 vs 浮動改裝 固定改裝（基礎改裝）：效果確定，達到等級即解鎖，管理者填入確切數值。
浮動改裝（高級改裝）：效果從可能池中隨機獲得，數值有範圍，模擬器中由使用者手動輸入已改裝的效果。
