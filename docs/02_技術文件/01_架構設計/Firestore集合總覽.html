型別系統
TypeScript Enum 枚舉化
所有分類字串欄位統一使用 TypeScript Enum，定義於 src/types/enums.ts。
📌 枚舉化欄位說明 職業、稀有度、模組槽位、裝甲類型、武器大類等分類欄位，均以 TypeScript Enum 定義。 Firestore 儲存 Enum 的字串值（如 "突擊手"），前端讀取後映射至對應 Enum。 詳細 Enum 定義見 Firebase 資料庫設計文件 §02。
Enum 名稱適用 Collection枚舉值（示例）
PilotClasspilots突擊手 / 狙擊手 / 戰術家 / 格鬥家 / 守護者 / 機械師
